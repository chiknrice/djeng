    Valid CodecFilter

    Project maintained by chiknriceTheme by mattgraham

    The class attribute of codec-filter should reference a valid implementation of org.chiknrice.djeng.CodecFilter

    Invalid Class

    The following configuration would result in java.lang.Integer is not a valid implementation of org.chiknrice.djeng.CodecFilter 

    <?xml version="1.0" encoding="UTF-8"?>
<config xmlns="http://www.chiknrice.org/djeng">

    <codecs>
        <codec-filter id="a-filter" class="java.lang.Integer"/>
        <element-codec id="string" class="org.chiknrice.djeng.fin.StringCodec"/>
        <composite-codec id="message" class="org.chiknrice.djeng.fin.PlainCompositeCodec"/>
    </codecs>
    <message-elements codec="message">
        <element index="1" codec="string"/>
    </message-elements>

</config>

    Non Exsiting Class

    The following configuration would result in Class not found org.chiknrice.djeng.NotExisting 

    <?xml version="1.0" encoding="UTF-8"?>
<config xmlns="http://www.chiknrice.org/djeng">

    <codecs>
        <codec-filter id="a-filter" class="org.chiknrice.djeng.NotExisting"/>
        <element-codec id="string" class="org.chiknrice.djeng.fin.StringCodec"/>
        <composite-codec id="message" class="org.chiknrice.djeng.fin.PlainCompositeCodec"/>
    </codecs>
    <message-elements codec="message">
        <element index="1" codec="string"/>
    </message-elements>

</config>
