    Message Elements Reference

    Project maintained by chiknriceTheme by mattgraham

    The message-elements codec attribute should reference a composite-codec.

    codec referencing an element-codec

    The following configuration would result in Key 'composite-codec-id-ref' with value 'string' not found for identity constraint of element 'config'. 

    <?xml version="1.0" encoding="UTF-8"?>
<config xmlns="http://www.chiknrice.org/djeng">

    <codecs>
        <element-codec id="string" class="org.chiknrice.djeng.fin.StringCodec"/>
        <composite-codec id="message" class="org.chiknrice.djeng.fin.PlainCompositeCodec"/>
    </codecs>
    <message-elements codec="string">
        <element index="1" codec="string"/>
    </message-elements>

</config>

    codec referencing a non existing composite-codec

    The following configuration would result in Key 'composite-codec-id-ref' with value 'x' not found for identity constraint of element 'config'. 

    <?xml version="1.0" encoding="UTF-8"?>
<config xmlns="http://www.chiknrice.org/djeng">

    <codecs>
        <element-codec id="string" class="org.chiknrice.djeng.fin.StringCodec"/>
        <composite-codec id="message" class="org.chiknrice.djeng.fin.PlainCompositeCodec"/>
    </codecs>
    <message-elements codec="x">
        <element index="1" codec="string"/>
    </message-elements>

</config>
