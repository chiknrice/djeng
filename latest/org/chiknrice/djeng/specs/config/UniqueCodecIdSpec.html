    Unique Codec ID

    Project maintained by chiknriceTheme by mattgraham

    The id attribute of a codec within the codecs should be unique.

    The following configuration would result in Duplicate unique value [x] declared for identity constraint of element "codecs". 

    <?xml version="1.0" encoding="UTF-8"?>
<config xmlns="http://www.chiknrice.org/djeng">

    <codecs>
        <element-codec id="x" class="org.chiknrice.djeng.fin.StringCodec"/>
        <composite-codec id="x" class="org.chiknrice.djeng.fin.PlainCompositeCodec"/>
    </codecs>
    <message-elements codec="x">
        <element index="1" codec="x"/>
    </message-elements>

</config>
