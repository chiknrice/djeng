    Valid Namespace

    Project maintained by chiknriceTheme by mattgraham

    The core namespace that needs to be defined in the xml is http://www.chiknrice.org/djeng. Failing to declare the namespace would cause the validation to fail.

    The following configuration would result in cvc-elt.1: Cannot find the declaration of element 'config'. 

    <?xml version="1.0" encoding="UTF-8"?>
<config>

    <codecs>
        <element-codec id="string" class="org.chiknrice.djeng.fin.StringCodec"/>
        <composite-codec id="message" class="org.chiknrice.djeng.fin.PlainCompositeCodec"/>
    </codecs>
    <message-elements codec="message">
        <element index="1" codec="string"/>
    </message-elements>

</config>
