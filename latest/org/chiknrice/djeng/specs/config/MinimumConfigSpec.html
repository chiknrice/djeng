    Minimum Configuration

    meet this

    Project maintained by chiknriceTheme by mattgraham

    The configuration requires at least 2 codecs, one composite codec and an element codec. The message elements should have at least one child element.

    No Composite Codec

    The following configuration would result in cvc-complex-type.2.4.b: The content of element 'codecs' is not complete. One of '{"http://www.chiknrice.org/djeng":element-codec, "http://www.chiknrice.org/djeng":composite-codec}' is expected. 

    <?xml version="1.0" encoding="UTF-8"?>
<config xmlns="http://www.chiknrice.org/djeng">

    <codecs>
        <element-codec id="string" class="org.chiknrice.djeng.fin.StringCodec"/>
    </codecs>
    <message-elements codec="string">
        <element index="1" codec="string"/>
    </message-elements>

</config>

    No Element Codec

    The following configuration would result in cvc-complex-type.2.4.a: Invalid content was found starting with element 'composite-codec'. One of '{"http://www.chiknrice.org/djeng":filter-codec, "http://www.chiknrice.org/djeng":element-codec}' is expected. 

    <?xml version="1.0" encoding="UTF-8"?>
<config xmlns="http://www.chiknrice.org/djeng">

    <codecs>
        <composite-codec id="message" class="org.chiknrice.djeng.fin.PlainCompositeCodec"/>
    </codecs>
    <message-elements codec="message">
        <element index="1" codec="message"/>
    </message-elements>

</config>

    No Element

    The following configuration would result in cvc-complex-type.2.4.b: The content of element 'message-elements' is not complete. One of '{"http://www.chiknrice.org/djeng":element, "http://www.chiknrice.org/djeng":composite}' is expected. 

    <?xml version="1.0" encoding="UTF-8"?>
<config xmlns="http://www.chiknrice.org/djeng">

    <codecs>
        <element-codec id="string" class="org.chiknrice.djeng.fin.StringCodec"/>
        <composite-codec id="message" class="org.chiknrice.djeng.fin.PlainCompositeCodec"/>
    </codecs>
    <message-elements codec="message">
    </message-elements>

</config>

    The Minimum

    The following configuration would be successful. 

    <?xml version="1.0" encoding="UTF-8"?>
<config xmlns="http://www.chiknrice.org/djeng">

    <codecs>
        <element-codec id="string" class="org.chiknrice.djeng.fin.StringCodec"/>
        <composite-codec id="message" class="org.chiknrice.djeng.fin.PlainCompositeCodec"/>
    </codecs>
    <message-elements codec="message">
        <element index="1" codec="string"/>
    </message-elements>

</config>
