    Valid ElementCodec

    Project maintained by chiknriceTheme by mattgraham

    The class attribute of element-codec should reference a valid implementation of org.chiknrice.djeng.ElementCodec

    The following configuration would result in java.lang.String is not a valid implementation of org.chiknrice.djeng.ElementCodec 

    <?xml version="1.0" encoding="UTF-8"?>
<config xmlns="http://www.chiknrice.org/djeng">

    <codecs>
        <element-codec id="string" class="java.lang.String"/>
        <composite-codec id="message" class="org.chiknrice.djeng.fin.PlainCompositeCodec"/>
    </codecs>
    <message-elements codec="message">
        <element index="1" codec="string"/>
    </message-elements>

</config>
