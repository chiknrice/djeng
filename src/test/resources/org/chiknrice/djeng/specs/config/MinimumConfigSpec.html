Minimum Configuration

The configuration requires at least 2 codecs, one composite codec and an element codec. The message elements also requires to be pointing to a composite codec and should have at least one child element.

Building a configuration with no composite codec: 

    


    

        Would result in cvc-complex-type.2.4.b: The content of element 'codecs' is not complete. One of '{"http://www.chiknrice.org/djeng":element-codec, "http://www.chiknrice.org/djeng":composite-codec}' is expected.

Building a configuration with no element codec: 

    


    

        Would result in cvc-complex-type.2.4.b: The content of element 'codecs' is not complete. One of '{"http://www.chiknrice.org/djeng":element-codec, "http://www.chiknrice.org/djeng":composite-codec}' is expected.

Building a configuration with no element: 

    


    

        Would result in cvc-complex-type.2.4.b: The content of element 'message-elements' is not complete. One of '{"http://www.chiknrice.org/djeng":element, "http://www.chiknrice.org/djeng":composite}' is expected.
