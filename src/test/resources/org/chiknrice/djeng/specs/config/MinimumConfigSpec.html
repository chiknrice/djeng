Minimum Configuration

meet this

The configuration requires at least 2 codecs, one composite codec and an element codec. The message elements also requires to be pointing to a composite codec and should have at least one child element.

No Composite Codec

The following configuration would result in cvc-complex-type.2.4.b: The content of element 'codecs' is not complete. One of '{"http://www.chiknrice.org/djeng":element-codec, "http://www.chiknrice.org/djeng":composite-codec}' is expected.

No Element Codec

The following configuration would result in cvc-complex-type.2.4.a: Invalid content was found starting with element 'composite-codec'. One of '{"http://www.chiknrice.org/djeng":filter-codec, "http://www.chiknrice.org/djeng":element-codec}' is expected.
