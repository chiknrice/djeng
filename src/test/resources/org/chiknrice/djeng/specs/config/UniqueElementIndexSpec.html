Unique Element Index

A sub-element's index attribute should be unique within the message-elements as well as any composite. Both element and composite are considered a sub-element.

Two elements under message-elements are having the same index

The following configuration would result in Duplicate unique value [1] declared for identity constraint of element "message-elements".

An element and a composite under message-elements are having the same index

The following configuration would result in Duplicate unique value [4] declared for identity constraint of element "message-elements".

Two elements under composite are having the same index

The following configuration would result in Duplicate unique value [2] declared for identity constraint of element "composite".

An element and a composite under another composite are having the same index

The following configuration would result in Duplicate unique value [3] declared for identity constraint of element "composite".
